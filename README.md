# Agent and Machine Exchange Standard (AMES)

An open standard for cooperative exchange between web publishers and
automated agents: structure-preserving translation of web content, with
machine-readable usage permissions built in.

## Specification

The canonical specification is [AMES_SPEC_v0_9.md](AMES_SPEC_v0_9.md).

## Status

v0.9 RFC. Stable enough for implementation and feedback. Breaking
changes before 1.0 will be clearly marked.

Worked examples, schema, and full documentation at
[openames.org](https://openames.org) are forthcoming.
